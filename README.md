# 待办清单 / Todone

一个基于Jetpack Compose构建的实用类应用。

## 主要界面

- 侧边导航抽屉：包含所有顶级界面的导航按钮。
- 今日：显示当天的待办项。
- 待办：包含所有的待办项，以列表及子项的形式显示。
- 检查单：显示所有的检查单项。
- 笔记：显示所有的笔记项。
- 设置：调整应用的首选项。
- 关于：显示一个对话框，包含应用图标、版本、作者信息。

## 次级界面
### 检查单编辑界面
通过点击 “+” 或者检查单卡片进入。添加检查单时，弹出的对话框要求您必须输入检查单的名字，否则将不允许添加检查单。点击已有检查单卡片将会进入编辑模式。

界面可以对检查单重命名，添加/删除检查单子项，以及删除或分享检查单。

### 笔记编辑界面

进入方式与检查单编辑界面类似。界面可以分享和删除笔记。如果笔记的标题和内容均为空，退出时将删除笔记。

### 设置界面
调整应用的通知，语言等。如果您连接了Android Studio，您可以使用Debug内的功能。
